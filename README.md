# Seldorna Kullie | Portfolio

This repository contains the source code for my personal portfolio website.  
It showcases my work as a **WordPress Developer & Web Designer**, highlighting projects, skills, and experience in building responsive, user‑focused websites.

## 🚀 Features
- Clean, responsive design built with HTML, CSS, and JavaScript
- Showcases WordPress, WooCommerce, and React projects
- Contact section with integrated form and social links
- Optimized layout for both desktop and mobile devices

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript, React
- **CMS:** WordPress, WooCommerce, Elementor
- **Tools:** Git, Firebase, Postman

## 📂 Sections
- **Home:** Introduction, resume link, and hero section
- **About:** Background, skills, and education
- **Projects:** Selected portfolio projects with descriptions and tech stacks
- **Contact:** Contact info, social links, and form
